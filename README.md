  ## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> pkg update && pkg upgrade 
> pkg install nodejs
> pkg install tesseract 
> git clone https://github.com/SofyanAMV09/ChikaBotv2
> cd ChikaBotv2
> bash install.sh
```

### Usage
```bash
> npm start
```
